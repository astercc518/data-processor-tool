## 🚀 快速使用

### 克隆项目

```bash
git clone https://github.com/astercc518/data-processor-tool.git
cd data-processor-tool
```

### 本地运行

```bash
pip install -r requirements.txt
python app.py
```

### Docker部署

```bash
# Windows
deploy.bat deploy

# 或使用Docker Compose
docker-compose up -d
```
